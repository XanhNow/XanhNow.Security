# XanhNow.Security

REST API bao mat trung tam cua he thong XanhNow.

## Ranh gioi bat buoc
- Goi Auth_Login_App, JWT_Refresh_Token_App, Passkey_Provider_App va SmartOtp_App qua private REST/gRPC contract.
- Khong thay the bon app con.
- Khong luu password, refresh token that, passkey private key hoac TOTP secret.
- UserId la dinh danh xuyen suot; so dien thoai chi la thong tin dang nhap.
